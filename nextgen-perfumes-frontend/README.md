# NextGen Perfumes Website

Welcome to the NextGen Perfumes website project! This project is designed to showcase our exquisite range of perfumes and provide a platform for customer interaction through reviews and contact options.

## Project Structure

The project is organized as follows:

```
nextgen-perfumes-website
├── src
│   ├── index.html        # Main landing page
│   ├── contact.html      # Contact form page
│   ├── reviews.html      # Reviews submission and display page
│   ├── css
│   │   └── styles.css    # Styles for the website
│   ├── js
│   │   └── main.js       # JavaScript functionality
│   └── assets
│       └── fonts
│           └── .gitkeep  # Keeps fonts directory in version control
├── README.md             # Project documentation
```

## Features

- **Landing Page**: The main page introduces NextGen Perfumes and provides navigation to contact and reviews sections.
- **Contact Form**: Users can reach out via email or phone, and the company's address is displayed for convenience.
- **Reviews Section**: Customers can leave reviews and ratings for our perfumes, enhancing community engagement.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd nextgen-perfumes-website
   ```
3. Open the `src/index.html` file in your web browser to view the website.

## How to Contribute

We welcome contributions! If you'd like to contribute to the NextGen Perfumes website, please follow these steps:

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them.
4. Push your branch to your forked repository.
5. Submit a pull request detailing your changes.

Thank you for your interest in contributing to NextGen Perfumes!
